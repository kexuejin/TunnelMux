# Code of Conduct

This project follows a Contributor Covenant style policy:

- Be respectful and constructive.
- Focus criticism on ideas, not people.
- Assume good intent and collaborate in good faith.
- Harassment, discrimination, or abusive behavior is not tolerated.

Maintainers may remove comments, issues, or contributions that violate these expectations.

For serious incidents, contact the maintainers privately through repository security contact channels.
