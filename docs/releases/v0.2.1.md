# TunnelMux v0.2.1

TunnelMux `v0.2.1` adds a minimal system tray integration for the desktop GUI.

## Highlights

- Close the main window to hide it instead of quitting.
- Always-on tray icon with:
  - `Show/Hide Window`
  - `Quit`
- Left-click tray toggle on macOS and Windows (menu remains the fallback on Linux).
